## ğŸš€ How PharmaCare Pro Works (Operational Flow)

The system is designed to follow the natural flow of your business:

1.  **Setting Up:** An Administrator adds the store details and staff accounts.
2.  **Stocking:** The Pharmacist or Admin enters new products into the **Inventory**, setting prices and reorder levels.
3.  **Selling:** The Dispenser (Cashier) uses the **New Sale** screen to process customer orders. The system automatically reduces the stock.
4.  **Reporting:** At any time, the Owner can check the **Dashboard** or **Reports** to see how the business is performing.
5.  **Refunding:** If a mistake is made, a refund can be initiated and approved by a manager to keep the books accurate.

## ğŸ› ï¸ Simple Installation Guide (Get Started in 5 Minutes)

You don't need to be an IT expert to use PharmaCare Pro. Follow these simple steps:

### Option 1: Using the Online Link (Easiest)
1.  Open your internet browser (Chrome, Edge, or Safari).
2.  Enter your unique **PharmaCare Web Link** (provided by your administrator).
3.  Log in with your email and password. That's it!

### Option 2: Running Locally (For Developers or Offline Use)
*If you need to run the code on your personal computer:*
1.  **Install Node.js:** Download it from [nodejs.org](https://nodejs.org/) (use the "LTS" version).
2.  **Download the Code:** Unzip the project folder onto your computer.
3.  **Open Terminal:** Right-click inside the folder and select "Open in Terminal" or "Command Prompt."
4.  **Type these commands:**
    ```bash
    npm install       # Wait for it to finish (installs the app)
    npm run dev       # Starts the app
    ```
5.  The terminal will show a link like `http://localhost:5173`. Hold **Ctrl** and click that link to open PharmaCare Pro.
